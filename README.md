# zomboid-web-control-panel
Extremly basic zomboid control panel


## setup
1. go to the folder
2. run `pip install -r requirements.txt`
3. go to static/file_work.py and edit line number 9 to refrence your server's whitelist.db
4. server runs with a default port of 5000, you need to make sure it's open or change it in the `web.py`
5. Make sure zomboid server is running in tmux using "zomboid" as a session name

# Disclaimer 
This project is made mostly for my personal use, it doesn't have all the features people might need. I made it in my free time, so that my friends can execute basic commands and restart server when needed.
